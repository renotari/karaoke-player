# Quick Start Guide

Get up and running with Karaoke Player in 5 minutes!

## Installation

1. **Download** the installer: `KaraokePlayer-Setup-1.0.0.exe`
2. **Run** the installer and follow the wizard
3. **Launch** Karaoke Player from the desktop shortcut or Start Menu

## First Launch

### Step 1: Select Your Media Directory

On first launch, you'll see a welcome dialog:

1. Click **Browse** to select your karaoke/music folder
2. Or use the default: `C:\Users\[YourName]\Music`
3. Click **OK**

### Step 2: Wait for Initial Scan

The app will scan your media directory:
- **Quick scan** indexes all files (a few seconds)
- **Background processing** extracts metadata and thumbnails
- You can start using the app immediately!

## Basic Operations

### Playing Your First Song

1. **Search**: Type a song name or artist in the search box
2. **Add**: Double-click a song to add it to the playlist
3. **Play**: Click the Play button or press `Space`

That's it! The song will start playing.

### Building a Playlist

**Add Multiple Songs**:
- Double-click songs to add them to the playlist
- Or drag songs from the catalog to the playlist pane

**Reorder Songs**:
- Drag songs up or down in the playlist

**Remove Songs**:
- Click the X button next to a song
- Or select and press `Delete`

### Essential Keyboard Shortcuts

| Action | Shortcut |
|--------|----------|
| Play/Pause | `Space` |
| Next Song | `Right Arrow` |
| Previous Song | `Left Arrow` |
| Volume Up/Down | `Up/Down Arrow` |
| Search | `Ctrl+F` |
| Fullscreen | `F11` |

## Display Modes

### Single Screen Mode (Default)

Everything in one window:
- Video player in the center
- Song catalog on the left
- Playlist on the right

**Maximize Video**: Click the maximize button to enter Video Mode with a collapsible control handle.

### Dual Screen Mode

Perfect for multi-monitor setups:
1. Press `Ctrl+D` to switch modes
2. Drag the Playback Window to your second monitor
3. Control everything from the main window

## Advanced Features

### Enable Crossfade

For smooth transitions between songs:

1. Press `Ctrl+,` to open Settings
2. Go to **Audio** tab
3. Check **Enable Crossfade**
4. Set duration (1-20 seconds)
5. Click **Apply**

### Playlist Composer

Build playlists efficiently:

1. Press `Ctrl+P` to open Playlist Composer
2. Browse your catalog on the left
3. Multi-select songs (`Ctrl+Click`)
4. Click **Add Selected**
5. Reorder as needed
6. Click **Save & Load for Play**

### Audio Visualizations

For MP3 files, choose a visualization style:

1. Open Settings (`Ctrl+,`)
2. Go to **Display** tab
3. Select **Visualization Style**:
   - Bars (default)
   - Waveform
   - Circular
   - Particles

## Tips & Tricks

### Quick Add to Playlist
- **Add Next**: `Ctrl+Shift+A` (plays after current song)
- **Add to End**: `Ctrl+A` (adds to end of queue)

### Search Tips
- Type partial names: "beat" finds "Beatles"
- Search by artist or title
- Recent searches appear in dropdown

### Playlist Management
- **Shuffle**: `Ctrl+S`
- **Clear All**: `Ctrl+L` (with confirmation)
- **Save Playlist**: File menu > Save Playlist

### Performance
- Let background processing complete for best performance
- Thumbnails load as you scroll
- Search is instant even with 30,000 files

## Common Questions

**Q: Where are my settings stored?**  
A: `%APPDATA%\KaraokePlayer\settings.json`

**Q: Can I change the media directory later?**  
A: Yes! Settings > General > Media Directory

**Q: How do I select a different audio output?**  
A: Settings > Audio > Audio Output Device

**Q: The video is too small/large**  
A: Use Video Mode (maximize button) or Fullscreen (`F11`)

**Q: Can I customize keyboard shortcuts?**  
A: Yes! Settings > Keyboard tab
